# V2X Communication System for Autonomous Vehicles

**Published Research** — icSoftComp Vietnam 2025 (Non-Springer track)  
"Real-Time V2X Communication System for Autonomous Vehicle Coordination and Safety"

## What it does
Simulates real-time Vehicle-to-Everything (V2X) data exchange between distributed nodes for intelligent transport scenarios.

## Key Features
- Real-time event-driven communication between nodes
- Message broadcasting logic with alert-trigger mechanisms
- Latency-aware processing for reliable synchronized communication
- 5G Sidelink with Spectrum-Aware Anti-Jamming and Precise Ranging

## Tech Stack
`Python` `5G Sidelink` `Real-time Systems` `Distributed Computing`
